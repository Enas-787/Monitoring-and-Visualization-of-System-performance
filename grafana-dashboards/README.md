# Grafana Dashboards

This folder contains Grafana dashboard exports and visualization configurations.

## Contents:
- Dashboard JSON export files
- Data source configuration templates
- Custom panel setups and queries
- Visualization templates and examples
- Alert configuration files

## Import Instructions:
1. Download .json files from this folder
2. In Grafana, navigate to Dashboards â†’ Import
3. Upload the JSON file or paste content directly
4. Configure data sources as needed
5. Adjust queries and thresholds as appropriate

## Team Guidelines:
- Export dashboards as JSON files with descriptive names
- Include version information in filenames
- Document any custom queries or data sources used
- Provide screenshots in the screenshots folder

## Naming Convention:
Format: `YYYY-MM-DD_dashboard-name_version.json`
Example: `2025-10-29_network-security-overview_v1.json`
