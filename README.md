# Digital Forensics & Cybersecurity Team Project

**Team Members:** Enas Alnaseri, Muhammad (alosaya), Naga (naginibvs)  
**Institution:** University at Albany - SUNY  
**Course:** Digital Forensics & Cybersecurity  

## Project Overview
Collaborative repository for our digital forensics analysis, cybersecurity monitoring, and data visualization using Python, Netdata, and Grafana.

## Repository Structure
- **`python-scripts/`** - Forensic analysis and cybersecurity automation tools
- **`netdata-configs/`** - System monitoring and performance configurations
- **`grafana-dashboards/`** - Data visualization and dashboard exports
- **`screenshots/`** - Visual documentation and evidence
- **`documentation/`** - Project reports, analysis, and meeting notes
- **`data-samples/`** - Safe, sanitized sample data for testing

## Getting Started
1. Review the README in each folder for specific guidelines
2. Follow security protocols when uploading any content
3. Use descriptive commit messages for all uploads
4. Coordinate with team members before major changes
