# Screenshots and Visual Documentation

This folder contains screenshots, diagrams, and visual evidence from our project work.

## Organization Guidelines:
- Use descriptive filenames with dates when relevant
- Naming format: `YYYY-MM-DD_tool-name_description.png`
- Example: `2024-10-29_netdata_dashboard_overview.png`
- Include brief descriptions in commit messages

## Supported Formats:
- **PNG** - Preferred for screenshots (best quality)
- **JPG** - Acceptable for photographs and complex images
- **PDF** - For multi-page reports and documentation

## Critical Security Note:
⚠️ **MANDATORY REVIEW BEFORE UPLOAD**
- Remove or blur any sensitive information (IP addresses, usernames, passwords)
- No real system credentials or proprietary information
- Only include sanitized examples appropriate for academic work
- When in doubt, consult with course instructors
